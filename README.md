# alpine-variations

Minimal, secure and regularly updated Alpine Linux Docker image variations with useful tools pre-installed.

## Available Variations
Choose the variation that fits your needs. All images follow below pattern:
```
aagjalpankaj/alpine:{{version}}-{{variation}}
```
